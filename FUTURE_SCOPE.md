# Future Scope

This file lists areas that are missing, lightly covered, or worth expanding into future tutorials and interview-practice sets.

## High-priority interview additions

- More window-function drills beyond the new chapter: harder `LAG`, `LEAD`, running total, moving average, and tie-handling problems.
- More Top-N per group variations, such as top 3 salaries by department with and without ties.
- Consecutive-days and streak problems.
- Gaps and islands problems.
- Median, percentile, and mode-style questions.
- Recursive CTEs for hierarchy problems.
- More `EXISTS` vs `IN` vs `JOIN` comparison problems.
- Advanced date problems: month-over-month growth, retention, cohort analysis, rolling 7-day metrics.
- Duplicate detection and deletion with safe MySQL patterns.
- Pivot-style reporting using conditional aggregation.

## MySQL tutorial additions

- Installation and setup guide for MySQL Workbench, CLI, and Docker.
- A clean sample database with repeatable seed data.
- Separate exercises with expected output tables.
- Transactions and isolation levels: dirty reads, non-repeatable reads, phantom reads.
- Index design: single-column, composite, covering indexes, and leftmost-prefix rule.
- `EXPLAIN` basics for query performance.
- Normalization vs denormalization with examples.
- Stored procedures, functions, triggers, and events with clearer use cases.
- Views vs CTEs vs temporary tables.
- Backup and restore in MySQL using `mysqldump`.
- User management, roles, grants, and privileges.

## Practice set ideas

- LeetCode 50 SQL study-plan solutions grouped by topic.
- Placement round mock tests: easy, medium, hard.
- Schema-design exercises for ecommerce, college, hospital, and HR systems.
- Debug-the-query exercises with common mistakes.
- Query optimization exercises using `EXPLAIN`.

## Quality improvements

- Add a `schema.sql` and `seed.sql` so every example can be run from scratch.
- Add screenshots or expected outputs for important examples.
- Add a glossary for SQL keywords and MySQL-specific terms.
- Expand `INTERVIEW_CHECKLIST.md` with mock-test links and common answer templates.
- Add links to official MySQL documentation for each advanced topic.
